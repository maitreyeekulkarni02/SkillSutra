SkillSutra is an intelligent web-based interview preparation platform designed for
students and fresh graduates. It provides AI-driven mock interviews, resume analysis,
role-based interview questions, aptitude practice, and detailed performance feedback.

## Features
- AI-based mock interview simulation
- Resume analysis and role suggestion
- Role and company-specific interview questions
- Performance feedback report
- Leaderboard-based motivation system

## Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: Python (Flask)  
Database: JSON (Prototype)  
Tools: GitHub, VS Code

## Future Scope
- WebRTC based video interviews
- Advanced AI/ML scoring
- Recruiter dashboard
